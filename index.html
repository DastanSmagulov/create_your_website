Программа курса

6 недель интенсивногооо обучения!

  • Введение в веб, основы HTML

    Инструменты для создания сайтов. Главный язык веба HTML. Теги, и аттрибуты. Структура. Проект Кофе Магазин.

  • Основы CSS

    Язык разметки CSS. Контейнеры. Работа со шрифтами, и цветом. Отступы. Псевдоэлементы. Свойство display. Проект: уникальная форма в css.

  • Figma, и верстка макета

    Знакомство с Фигмой. Flexbox. Свойство position. Верстка настоящего макета с наставлением преподователя. Закрепление материала.

  • Завершение верстки макета. Ответы на вопросы

    Небольшая пауза для уверенного усвоения материала. Двухчасовые индивидуальные уроки с добиванием пробелов каждого из учеников, рефлексия. Готовый проект.

  • Знакомство с библиотекой anima js. Основа адаптивной верстки

    Свойство transform. Анимация в css. Умение читать документацию, и подключать библиотеки. Как сделать так, чтобы сайт выглядел одинаково хорошо на всех устройствах. Где искать вдохновения для своего уникального проекта, и почему воровать дизайны не плохо.

  • Завершение курса. Презентации проектов. Хостинг github pages

    Индивидульный урок, помощь с проектом для каждого ученика. Презентация проектов. Хостинг github pages, и возможность поделиться своим сайтом, и прогрессом на курсе. Наставления преподователя, и ответы на вопросы.
